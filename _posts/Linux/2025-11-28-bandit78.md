# Level 7 → Level 8
![bandit78](/image/2025-11-28-bandit78/bandit78.png)

먼저 현재 디렉터리에서 data.txt 파일이 있는지 확인하였다.

![data.txt파일확인](/image/2025-11-28-bandit78/data.txt파일확인.png)

`cat` 명령어를 사용해서 `data.txt` 파일 내용을 확인해보았는데 수많은 단어들을 확인할 수 있었다.

![cat data.txt](/image/2025-11-28-bandit78/cat data.txt.png)

Level 8로 이동하기 위한 비밀번호는  "millionth"단어 옆에 저장되어 있으므로 grep 명령어를 사용해야 한다.

```c
grep 명령어는 텍스트 파일에서 특정 문자열을 검색하여 해당 문자열이 포함된 줄을 출력하는 리눅스 명령어이다.
```
![비밀번호 확인](/image/2025-11-28-bandit78/비밀번호 확인.png)

```c
grep "millionth" data.txt
```
위 명령어를 입력하여 비밀번호를 확인할 수 있었다.



grep 명령어 형식
```c
grep [옵션] [검색할 문자열] [파일명]
```
