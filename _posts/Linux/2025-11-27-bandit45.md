# Level 4 → Level 5
![bandit45](/image/2025-11-27-bandit45/bandit45.png)

먼저 현재 디렉터리에서 파일 목록을 확인한 뒤, inhere 디렉터리가 존재하는 것을 확인하였다.

![파일목록확인](/image/2025-11-27-bandit45/파일목록확인.png)

이후 cd 명령어를 사용하여 inhere 디렉터리로 이동하고, 파일 목록을 확인하였다.

![경로 이동 후 파일목록 출력](/image/2025-11-27-bandit45/경로 이동 후 파일목록 출력.png)

여러 개의 파일 중 사람이 읽을 수 있는 파일을 찾기 위해 `file ./-file*` 명령어를 사용하였다.

![file명령어사용](/image/2025-11-27-bandit45/file명령어사용.png)

```c
file 명령어는 파일의 내용을 분석하여 텍스트 파일인지 바이너리 파일인지 등 파일 형식을 확인하는 리눅스 명령어이다.
  
-file* -file로 시작하는 모든 파일
```

명령어 실행 결과 -file07이 ASCII text 형식의 파일로 확인되었으며, 해당 파일이 사람이 읽을 수 있는 파일임을 확인하였다.

이후 파일 -file07에서 비밀번호를 확인할 수 있었다.

![비밀번호 확인](/image/2025-11-27-bandit45/비밀번호 확인.png)
