# Level 0
![bandit](/image/2025-11-24-bandit0/bandit.png)
Bandit Level 0 문제에서는 SSH를 이용하여 OverTheWire 서버에 접속해야 한다.

문제 설명에 접속할 호스트 주소, 사용자 이름, 포트 번호가 제공되어 있으므로 SSH 명령어를 이용해 해당 서버에 접속하면 된다.

SSH 기본 명령어 형식은 다음과 같다.
```c
ssh 사용자@서버주소 -p 포트번호
```

적용한 명령어
```c
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
![ssh](/image/2025-11-24-bandit0/ssh.png)

입력하면 password 입력창이 나오게 되는데 주어진 비밀번호 bandit0을 입력하면 접속에 성공하게 된다.

![접속완료화면](/image/2025-11-24-bandit0/접속완료화면.png)
