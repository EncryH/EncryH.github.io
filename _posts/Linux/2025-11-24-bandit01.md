# Level 0 → Level 1
![bandit01](/image/2025-11-24-bandit01/bandit01.png)

먼저 홈 디렉터리에 readme이 있는지 확인한다.

```c
ls -l 은 리눅스에서 파일과 디렉터리 정보를 자세하게 보여주는 명령어
```

![ls -l](/image/2025-11-24-bandit01/ls -l.png)

readme 파일에는 Level 1로 이동하기 위한 비밀번호가 저장되어 있다.
파일 내용을 확인하기 위해 `cat` 명령어를 사용한다.

```c
cat readme
```

![cat](/image/2025-11-24-bandit01/cat.png)

출력된 비밀번호
```c
ZjLjTmM6FvvyRnrb2rfNWOZOTa6ip5If
```

Level 1 접속
```c
ssh bandit1@bandit.labs.overthewire.org -p 2220
```

exit 명령어를 사용해서 접속되어 있는 bandit0에서 로그아웃하고 접속해야 함.
