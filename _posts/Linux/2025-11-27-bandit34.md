# Level 3 → Level 4
![bandit34](/image/2025-11-27-bandit34/bandit34.png)

먼저 파일 목록을 확인하여 `inhere` 디렉터리를 확인하였다.

이후 `cd` 명령어를 사용해 `inhere` 디렉터리로 이동한 뒤 내부 파일 목록을 확인하였다.

```c
cd(Change Directory) 명령어는 현재 작업 중인 디렉터리를 다른 디렉터리로 이동할 때 사용하는 리눅스 명령어이다.
```

![경로를inhere로이동](/image/2025-11-27-bandit34/경로를inhere로이동.png)

디렉터리 내 파일 목록을 확인한 후, 숨겨진 형태로 보이는 ...Hiding-From-You 파일을 발견하였다.

해당 파일의 내용을 확인하기 위해 리눅스의 cat 명령어를 사용하여 비밀번호를 확인할 수 있었다.

![cat명령어로비밀번호확인](/image/2025-11-27-bandit34/cat명령어로비밀번호확인.png)
