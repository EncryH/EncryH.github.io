# 테스트 게시글

이 글은 블로그 게시 여부를 확인하기 위해 작성한 테스트용 포스트입니다.

## 확인 내용

1. `_posts` 경로에 새 글이 정상적으로 추가되는지
2. 카테고리와 태그가 아카이브에 반영되는지
3. GitHub Pages 또는 Jekyll 빌드에서 문제없이 렌더링되는지

## 메모

현재 글은 텍스트만 포함한 가장 단순한 형태로 작성했습니다.
필요하면 이후에 이미지, 코드 블록, 목차, 카테고리 구조까지 함께 점검할 수 있습니다.

## 정리

테스트가 끝난 뒤에는 이 글을 유지하거나 삭제해서 운영 포스트만 남기면 됩니다.
