# 모의해킹을 위한 리눅스 실습 네트워크 구성

# 실습 네트워크 구성
![실습네트워크 구성](/image/2025-10-21-Linux/실습네트워크 구성.png)
모의 해킹과 이에 대응하는 서버를 구축하려면 상호 통신이 가능한 네트워크를 구성해야 한다.

이를 위해서 VirtualBox 설정에서 NAT 네트워크를 설정해야 하고, 이후에 칼리 리눅스와 우분투의 운영체제 네트워크 설정도 필요하다.

기본 설치 시 DHCP(유동 IP 할당)의 구성을 고정 IP로 변경하여 동일한 IP로 실습할 수 있는 환경을 만들 것이다.

## VirtualBox 네트워크 설정
![VirtualBox 네트워크 설정](/image/2025-10-21-Linux/VirtualBox 네트워크 설정.png)
새로운 NatNetwork 생성 후 NAT 네트워크 정보를 다음과 같이 수정
- 네트워크 이름 : TestNet
- 네트워크 CIDR : 192.168.100.0/24
- 네트워크 옵션 : DHCP 지원 체크

![VirtualBox Settings](/image/2025-10-21-Linux/VirtualBox Settings.png)
kali & ubuntu 리눅스 둘 다 네트워크 설정에서 NAT 네트워크의 TestNet을 선택한 후 확인
