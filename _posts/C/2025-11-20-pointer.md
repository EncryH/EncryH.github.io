# 포인터(Pointer)란?
포인터는 메모리 주소(Address)를 저장하는 변수이다.
일반 변수는 값을 저장하지만, 포인터 변수는 다른 변수의 위치(주소)를 가리킨다.

예를 들어 다음 코드에서
```c
int num = 10;
int* ptr = &num;
```
- num → 정수 값 저장
- &num → num의 메모리 주소
- ptr → num의 주소를 저장하는 포인터 변수

즉 포인터를 이용하면 특정 메모리 위치에 직접 접근할 수 있다.

# 포인터 연산 (Pointer Arithmetic)
포인터는 자료형 크기만큼 이동하는 특징이 있다.

```c
int a[2] = { 0x12345678, 0x87654321 };
int* ptr_a = a;

printf("pointer a : %p\n", ptr_a);
printf("pointer a + 1 : %p\n", ptr_a + 1);
printf("*(ptr_a + 1) : %x\n", *(ptr_a + 1));
```

출력
```c
pointer a : 0x1000
pointer a + 1 : 0x1004
*(ptr_a + 1) : 0x87654321
```

# 지역 변수와 메모리 배치
```c
int a1 = 0x1234;
int a2 = 0x5678;
```
위 코드에서 지역 변수는 메모리에 연속적으로 배치된다는 보장이 없다.

따라서
```c
*(ptr_a1 + 1)
```
이 다음 변수를 가리키는 것은 우연일 수 있으며 실행 환경에 따라 달라질 수 있다.

# 문자열과 포인터
```c
char str[] = "123456789";
char* ptr = str;

printf("%c\n", *ptr);
printf("%c\n", *(ptr + 1));
printf("%c\n", *(ptr + 2));
```

메모리 구조
```c
주소    값
1000   '1'
1001   '2'
1002   '3'
```
포인터는 한 문자씩 이동하며 접근한다.

# 2차원 배열과 메모리
2차원 배열도 실제 메모리에서 연속된 메모리 공간으로 저장된다.
```c
int array[][3] = {
    {3,2,1},
    {6,5,4},
    {9,8,7}
};
```
