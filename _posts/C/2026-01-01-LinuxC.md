# GCC란 무엇인가
GCC(GNU Compiler Collection)는 C, C++, Java 등 다양한 프로그래밍 언어를 컴파일할 수 있는 컴파일러이다.

리눅스 환경에서 C 프로그램을 컴파일할 때 가장 널리 사용된다.

GCC를 이용하면 C 소스 코드를 컴파일하여 실행 가능한 프로그램으로 변환할 수 있다.
